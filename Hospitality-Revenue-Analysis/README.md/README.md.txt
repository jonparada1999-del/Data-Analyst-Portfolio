Hospitality Demand & Revenue Analysis
Data Analyst Portfolio Project #5 | Python, SQL & Tableau

The Business Problem: Analyzed 119,000 hotel records to identify revenue drivers and cancellation risks for an international hotel chain.

Technical Proof: Utilized Python for data engineering, SQL (BigQuery) for cloud-based aggregation, and Tableau for executive storytelling.

The Key Results:

Identified Online TAs as the primary revenue engine, contributing over $16M.

Discovered a critical seasonal cancellation risk exceeding 60% during peak summer months.
